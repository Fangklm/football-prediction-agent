# ⚽ 足球赛果判断系统 (多联赛版)

[![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg)]()

一个基于专业赔率分析的足球赛果预测系统，整合多家知名博彩公司数据，支持13个主要联赛的深度分析规则。

## 🌟 功能特色

- **🏆 多联赛支持**: 涵盖五大联赛、亚洲联赛、南美联赛、北美联赛等13个主要联赛
- **🔍 专业规则分析**: 
  - 德乙联赛专用规则 (5大规则体系)
  - 日职联(J1)专用规则 (3大规则体系)
  - 日职乙(J2)专用规则 (3大规则体系)
  - 英冠专用规则 (上盘/下盘规则)
  - 法乙专用规则 (上盘/下盘规则)
  - 美职联专用规则 (上盘/下盘规则)
  - 其他联赛通用智能规则
- **📊 多维度赔率分析**: 整合澳门、威廉希尔、皇冠、立博等知名博彩公司的赔率数据
- **🎯 智能判断建议**: 基于规则触发情况和权重算法给出精确的投注建议
- **💡 交互式界面**: 友好的命令行界面，支持实时分析和历史数据管理

## 🚀 快速开始

### 系统要求
- Python 3.7+
- 无外部依赖 (纯Python实现)

### 安装与启动
```bash
# 克隆项目
git clone https://github.com/Fangklm/football-prediction-agent.git
cd football-prediction-agent

# 直接运行 (无需安装依赖)
python main.py
```

### 基本指令
| 指令 | 功能 | 示例 |
|------|------|------|
| `leagues` | 查看所有支持的联赛列表 | `leagues` |
| `quit` | 退出系统 | `quit` |
| `analyze [比赛编号]` | 分析指定比赛 | `analyze J1-001` |
| 直接回车 | 添加新比赛数据 | - |

## 📖 使用指南

### 添加比赛数据
1. **输入比赛编号**: 任意字符串标识 (建议格式: `联赛-序号`)
2. **选择联赛代码**: 从1-13中选择对应联赛
3. **输入赔率数据**: 按提示依次输入三家公司的赔率
   - 澳门初盘赔率 (胜 平 负)
   - 威廉希尔初盘赔率 (胜 平 负)  
   - 皇冠初盘赔率 (胜 平 负)

### 完整示例

## 🔬 核心分析规则

### 🇩🇪 德乙专用规则 (最完善)
| 规则类型 | 触发条件 | 建议方向 |
|----------|----------|----------|
| **上盘规则** | WL特定赔率组合、HG-WL差值≥0.08 | 上盘/低赔率方 |
| **下盘规则** | WL特定组合、HG-WL差值0.1-0.2、容让规则 | 下盘/高赔率方 |
| **低水规则** | 1.70 < AM < 1.89 | 低水方 |
| **高水不败** | AM>2.00且AM-WL=0.05 | 高水方不败 |
| **WL特定组合** | 1.80+3.70+4.20等特定组合 | 相应方向 |

### 🇯🇵 日职联(J1)专用规则
| 规则类型 | 核心逻辑 | 建议方向 |
|----------|----------|----------|
| **上盘规则** | HG-WL≥0.2、WL特定赔率(1.40,1.44,1.57,1.88) | 上盘/低赔率方 |
| **下盘规则** | WL-HG≥0.1且WLD<HGD、高水方规则 | 下盘/高赔率方 |
| **低水方规则** | AM>2.00且AM-WL=0.03 | 低水方 |

### 🇯🇵 日职乙(J2)专用规则  
| 规则类型 | 核心逻辑 | 建议方向 |
|----------|----------|----------|
| **上盘规则** | HG-WL=0.01(WLD≥HGD)、WL特定赔率 | 上盘/低赔率方 |
| **下盘规则** | 0.1<HG-WL<0.2、WL-HG≥0.3、WL特定组合 | 下盘/高赔率方 |
| **高指方规则** | 2.10<AM<2.19且AM>WL | 高指方 |

### 🏴󠁧󠁢󠁥󠁮󠁧󠁿 英冠专用规则
| 规则类型 | 触发条件 | 建议方向 |
|----------|----------|----------|

| **下盘规则** | AM>2.40、AM-WL差值规则 | 下盘/高赔率方 |

### 🇫🇷 法乙专用规则
| 规则类型 | 触发条件 | 建议方向 |
|----------|----------|----------|
| **上盘规则** | AM<1.70、WL特定赔率和组合 | 上盘/低赔率方 |
| **下盘规则** | 2.20<AM<2.49、WL特定赔率、AM-WL差值 | 下盘/高赔率方 |

### 🇺🇸 美职联专用规则
| 规则类型 | 触发条件 | 建议方向 |
|----------|----------|----------|
| **上盘规则** | HG-WL≥0.20、WL特定赔率、HGD-WLD差值 | 上盘/低赔率方 |
| **下盘规则** | HG-WL>0.5、WL特定赔率 | 下盘/高赔率方 |

## 📊 输出格式说明

### 分析结果包含:
1. **📋 比赛信息**: 联赛名称、各博彩公司赔率对比
2. **⚡ 触发规则**: 详细列出所有匹配的专业分析规则  
3. **🎯 综合判断**: 基于规则权重的智能建议
   - 🔥 **强烈建议**: 支持上盘/下盘
   - 🟡 **关注平局**: 平局概率较高
   - ⚖️ **谨慎判断**: 信号混合，存在分歧

### 智能判断逻辑
- **上盘信号**: 低赔率方、强队主导、上盘规则触发
- **下盘信号**: 高赔率方、弱队机会、下盘规则触发  
- **平局信号**: 平局赔率<3.0、特定平局组合
- **混合信号**: 多种信号并存，建议谨慎分析

## 🏗️ 技术架构

### 核心设计
- **🎯 面向对象**: 清晰的类结构，职责分离
- **🔧 模块化规则**: 每个联赛规则独立封装，易于维护
- **🚀 高扩展性**: 新增联赛和规则只需添加对应方法
- **🛡️ 容错设计**: 完善的异常处理和用户友好提示
- **⚡ 高性能**: 纯Python实现，无外部依赖，启动迅速

### 代码结构
```
main.py
├── FootballPredictionSystem (主类)
│   ├── __init__() - 初始化联赛和数据存储
│   ├── add_match() - 添加比赛数据
│   ├── analyze_match() - 核心分析方法
│   ├── analyze_*_rules() - 各联赛专用规则
│   ├── check_*_rules() - 具体规则检查方法
│   └── format_result() - 结果格式化输出
└── interactive_system() - 交互式界面
```
